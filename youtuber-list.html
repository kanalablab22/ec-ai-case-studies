YouTuberリストアップ自動化

YouTube APIを活用し、特定ジャンルのYouTuberチャンネルを自動でリストアップ。チャンネル登録者数・動画再生数・投稿頻度などのデータを自動収集し、PR案件の候補リストを効率的に作成。

カテゴリ データ収集・リスト作成
技術スタック Python / YouTube API / Google Sheets
収集規模 数百チャンネル
出力形式 CSV / Google Sheets
課題と解決策

😰 Before（課題）

  • YouTubeで1チャンネルずつ手動で検索・確認
  • 登録者数や再生数を手動でメモ
  • PR候補のリストアップに丸1日かかる

🎉 After（解決）

  • キーワード指定で関連チャンネルを自動収集
  • チャンネル統計データを自動取得・CSV出力
  • 数百チャンネルを数分でリスト化
